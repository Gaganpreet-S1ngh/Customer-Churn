# Customer Churn Prediction System

End-to-end ML pipeline to predict customer churn using Scikit-learn.

## Tech Stack
- Python
- Scikit-learn
- MLflow
- FastAPI
- Docker
- Prometheus

## Features
- Trained ML models on 50,000+ records
- Hyperparameter tuning with GridSearchCV
- REST API for predictions
- Dockerized deployment
- Monitoring with Prometheus
